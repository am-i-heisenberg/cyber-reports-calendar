# Cyber Security Report Calendar

A single-page reference showing **when major cybersecurity industry reports are released**, who publishes them, and links to the most recent editions.

## Reports Covered

| Category | Examples |
|---|---|
| Broad Threat & Breach | Verizon DBIR, CrowdStrike GTR, Mandiant M-Trends, IBM X-Force, Sophos, Unit 42, Microsoft DDR |
| OT / CNI | Dragos Year in Review, CISA ICS Advisories, E-ISAC GridEx |
| Insurance & Claims | Allianz Risk Barometer, Coalition Cyber Claims |
| UK Government | NCSC Annual Review, DSIT Cyber Security Breaches Survey |
| EU / International | ENISA Threat Landscape, WEF Cybersecurity Outlook, WEF Global Risks |

## Features

- **Calendar view**: month-by-month cards highlighting peak reading months (Jan–Mar, Oct)
- **List view**: grouped by report category with direct links
- Single HTML file — no build step, no dependencies
- Works on GitHub Pages out of the box

## Hosting on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your calendar will be live at `https://<username>.github.io/cyber-reports-calendar/`
